# MiniMax 图片生成器 - Web应用

一个现代化的Web应用程序，使用MiniMax AI API生成图片。

## ✨ 功能特点

- **文本转图片生成**：通过文本描述创建图片
- **多种AI模型支持**：支持 image-01 和 image-01-live 模型
- **画风控制**：多种画风选择（漫画、元气、中世纪、水彩）
- **多种图片比例**：支持 1:1, 16:9, 4:3, 3:2, 3:4, 9:16, 21:9
- **自定义尺寸**：可调整图片宽度和高度（仅 image-01）
- **批量生成**：一次性生成1-9张图片
- **角色参考**：上传人脸照片进行角色定制生成
- **图片预览**：网格视图和全屏预览
- **下载选项**：支持单张或批量下载
- **Prompt优化器**：可选启用AI驱动的提示词优化
- **响应式设计**：支持桌面和移动设备
- **中文界面**：完整的简体中文界面

## 🛠️ 技术栈

- **前端框架**: React 18 + TypeScript
- **构建工具**: Vite
- **样式**: Tailwind CSS v4
- **API**: MiniMax 图像生成 API

## 🚀 快速开始

### 环境要求

- Node.js 16+ 和 npm

### 安装步骤

1. **进入前端目录**：
```bash
cd frontend
```

2. **安装依赖**（已安装）：
```bash
npm install
```

3. **启动开发服务器**：
```bash
npm run dev
```

4. **打开浏览器访问**：`http://localhost:5173`

### 构建生产版本

```bash
npm run build
```

构建文件将生成在 `dist` 目录。

### 预览生产版本

```bash
npm run preview
```

## 📖 使用说明

### 1. 设置API密钥

1. 打开应用后，点击右上角的 **"⚠️ 请设置 API Key"** 按钮
2. 在弹出的面板中粘贴您的MiniMax API密钥
3. 点击 **"保存密钥"** 按钮
4. 按钮会变成绿色的 **"✓ 已设置 API Key"**

### 2. 获取API密钥

1. 访问 [MiniMax开放平台](https://platform.minimaxi.com)
2. 注册或登录您的账户
3. 进入"账户管理 > 接口密钥"
4. 复制API密钥并粘贴到应用中

### 3. 生成图片

1. **输入图片描述**：在文本框中输入您想要的图片内容
2. **调整设置**（可选）：
   - 选择AI模型（image-01 或 image-01-live）
   - 设置图片比例
   - 调整尺寸（仅image-01）
   - 选择画风（仅image-01-live）
   - 设置生成数量
3. **点击生成**：点击 **"🎨 生成图片"** 按钮
4. **等待完成**：生成过程可能需要几秒钟
5. **预览下载**：点击图片可全屏预览，支持下载或复制链接

### 4. 使用模板

点击 **"显示模板"** 按钮可以查看预设的提示词模板，快速开始生成。

### 5. 快捷键

- **Ctrl+Enter** (Windows) 或 **Cmd+Enter** (Mac)：快速生成图片

## ⚙️ 配置选项

### 模型选择

- **image-01**: 标准图片生成，支持自定义尺寸
- **image-01-live**: 增强模型，支持画风控制

### 图片比例

- 1:1 (正方形)
- 16:9 (宽屏)
- 4:3 (标准)
- 3:2 (照片)
- 3:4 (竖屏)
- 9:16 (手机竖屏)
- 21:9 (超宽屏)

### 画风类型（仅 image-01-live）

- 🎨 漫画 (Comic)
- ⚡ 元气 (Energetic)
- 🏰 中世纪 (Medieval)
- 🌊 水彩 (Watercolor)

### 其他选项

- **生成数量**: 1-9张
- **Prompt优化器**: 启用AI自动优化描述
- **AI水印**: 在图片中添加水印
- **随机种子**: 输入相同种子可生成相似图片

## 🎯 常见问题

### Q: 为什么点击"生成图片"没有反应？
A: 请确保已经设置了API密钥，且输入了图片描述。

### Q: 生成失败怎么办？
A: 检查API密钥是否正确，网络是否稳定，或者API配额是否用完。

### Q: 如何生成多张图片？
A: 在设置面板中调整"生成数量"滑块，可以一次生成1-9张图片。

### Q: 图片下载失败？
A: 某些浏览器可能阻止下载，请尝试右键点击图片选择"在新标签页打开"后下载。
